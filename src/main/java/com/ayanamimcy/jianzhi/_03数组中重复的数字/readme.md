# 题目

找出数组中重复的数字。

在一个长度为 n 的数组 nums 里的所有数字都在 0～n-1 的范围内。数组中某些数字是重复的，但不知道有几个数字重复了，也不知道每个数字重复了几次。请找出数组中任意一个重复的数字。

示例 1：

``` java
输入：
[2, 3, 1, 0, 2, 5, 3]
输出：2 或 3

```

限制：

2 <= n <= 100000

# 解法

1. 通过排序算法排序后进行统计
2. 通过hash表
3. 由于数字的范围都是n-1范围内的值，所以将对应的值放到对应的下标中，如果有存在相同的下标则值重复

由于题目是要求找到任意的一个重复的值，所以找到一个值直接返回即可.
